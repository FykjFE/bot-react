# react快速开发模板

> 代码提交规范

|     关键字     |   注释          |
|----------|-------------|
| feat     | 新特性/功能      |
| fix      | 修复bug       |
| docs     | 文档修改        |
| style    | 代码格式修改      |
| refactor | 代码重构,不添加新功能 |
| perf     | 提升性能        |
| test     | 代码测试        |

git message使用 `feat: 添加登录功能` 类似
