# react快速开发模板

> 代码提交规范

|     关键字     |   注释          |
|----------|-------------|
| feat     | 新特性/功能      |
| fix      | bug修复       |
| docs     | 仅修改文档        |
| style    | 代码格式的美化      |
| refactor | 代码重构(没有fix和feat) |
| perf     | 提升性能        |
| test     | 代码测试        |
| chore     | src目录以外的修改|
| ci     | CI/CD改动|

> 例子
```shell
git commit -m "feat: 添加登录功能"
```
